# XLiveServices

A library to consume Xbox LIVE and Games for Windows LIVE services.

## Installation

Add this line to your application's Gemfile:

    gem 'xlive_services'

And then execute:

    $ bundle

Or install it yourself as:

    $ gem install XLiveServices
